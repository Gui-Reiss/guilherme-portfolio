# 🚀 Portfólio — Guilherme Gonçalves

Este é o meu portfólio pessoal desenvolvido com **HTML5, CSS3 e JavaScript puro**, com foco em performance, responsividade e experiência do usuário.

🔗 **Acesse o projeto:**
👉 https://gui-reiss.github.io/guilherme-portfolio/

---

## ✨ Sobre o projeto

O objetivo deste projeto é apresentar minhas habilidades como desenvolvedor front-end através de uma aplicação leve, moderna e totalmente responsiva.

O portfólio foi construído sem o uso de frameworks, priorizando o domínio dos fundamentos da web.

---

## 🧠 Funcionalidades

* 📱 Layout **100% responsivo** (mobile-first)
* 🎨 Design moderno em **dark mode**
* ✨ Animações ao scroll com **Intersection Observer**
* 🍔 Menu mobile interativo (hamburger)
* 📊 Barras de habilidades animadas
* 📩 Formulário com validação em JavaScript
* 🔝 Botão “voltar ao topo”
* 🔗 Navegação suave entre seções
